Uppercase status pill for run state, smoothness verdicts, and tier labels. Tone carries meaning.

```jsx
<Badge tone="good" dot>Validated</Badge>
<Badge tone="warn">Best-effort</Badge>
<Badge tone="bad" dot>Flagged</Badge>
<Badge tone="brand">DLSS 3</Badge>
```

Tones: `neutral | good | warn | bad | info | brand | solid`. `dot` adds a leading status dot.
